# Prompt 07 — Estrutura Completa da Página

**Categoria:** Landing Page

**Modelo sugerido:** ChatGPT 4o

## Objetivo

Crie toda a estrutura e copy de uma landing page de alta conversão para seu SaaS.

## Prompt

Crie a estrutura completa de uma landing page de alta conversão para [NOME DO SAAS]. Inclua em ordem: Hero (headline + subheadline + CTA), Prova social (3 métricas fictícias mas realistas), Problema (seção de dor), Solução (como funciona em 3 passos), Benefícios (6 cards), Preço (1 plano simples), FAQ (5 perguntas), CTA final. Para cada seção escreva o copy completo em português.

# Prompt 13 — Script de Abordagem Fria

**Categoria:** Clientes

**Modelo sugerido:** ChatGPT 4o

## Objetivo

Escreva mensagens frias para DM, WhatsApp ou LinkedIn que geram curiosidade e resposta.

## Prompt

Escreva um script de mensagem fria para abordar potenciais clientes do [NOME DO SAAS] via [CANAL: Instagram DM / WhatsApp / LinkedIn]. A mensagem deve ter no máximo 5 linhas, não parecer spam, gerar curiosidade, e terminar com uma pergunta que incentive resposta. Crie 3 variações com abordagens diferentes.

# Prompt 04 — Gerar Nome + Domínio

**Categoria:** Identidade

**Modelo sugerido:** ChatGPT 4o

## Objetivo

Crie opções de nome memoráveis com domínios disponíveis para o seu SaaS brasileiro.

## Prompt

Crie 10 opções de nome para um SaaS brasileiro que resolve [PROBLEMA] para [PÚBLICO]. Critérios: fácil de pronunciar, memorável, máximo 2 sílabas ou palavras, que transmita [BENEFÍCIO PRINCIPAL]. Para cada nome sugira também um domínio .com.br ou .app disponível. Destaque os 3 melhores com justificativa.
